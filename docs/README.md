# Habit Tracker (Vanilla JS)

A simple habit tracker that lets you add habits, mark them done for the day, and track streaks. Data persists using `localStorage`.

## Features
- Add and manage habits
- Mark a habit done for today / undo
- Streak calculation (consecutive days ending today)
- Filter: Active / All / Archived
- Persistent storage via `localStorage`

## Run locally
Just open `index.html` in your browser.
